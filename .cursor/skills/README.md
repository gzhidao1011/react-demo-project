## 与 Rules 的关系

- **Rules** (`.cursor/rules/`)：提供持久化上下文和规范（总是应用）
- **Skills** (`.cursor/skills/`)：执行特定任务（按需触发）

Rules 和 Skills 互补：
- Rules 提供项目规范和标准
- Skills 提供具体任务的执行指导
