## 📝 文档说明

### 文档命名规范

- 格式：`<编号>-<主题>.mdc`
- 编号：按主题分类，从 00 开始递增
- 主题：使用中文描述文档主题

### 文档结构

每个文档通常包含：
- **核心原则**：文档的基本原则
- **详细规范**：具体的规范和最佳实践
- **示例代码**：实际使用示例
- **相关文档**：与其他文档的关联
- **常见问题**：FAQ 部分

### 文档更新

- 文档更新应遵循 [08-Git提交规范.mdc](./08-Git提交规范.mdc)
- 重大变更应通过 PR 审查
- 更新后应同步更新本文档索引
